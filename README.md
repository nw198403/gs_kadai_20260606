# つなぎノート - 交換日記チャットアプリ

## サイト概要
カウンセリングサービス向けの交換日記Webアプリです。  
クライアントがカレンダーから日付を選んで日記を書き、カウンセラーがコメントを返すことができます。  
LINEのようなチャット形式で、日付ごとのやり取りをわかりやすく確認できます。

---
## 工夫した点・こだわった点
- カレンダーUIを採用し、日付をクリックするとその日のやり取りがチャット形式で表示されます（過去のやり取りを日付で探しやすくするため）
- チャット画面はLINEを参考に、クライアントとカウンセラーのメッセージを左右に分けて表示し、直感的に読めるようにしました
- 白・ベージュ・薄い水色を基調カラーにし、ホッとするような温かみのあるデザインを意識しました
- Firebase Anonymous Authentication を使い、アカウント登録なしで名前とロールを選ぶだけで使い始められるシンプルな仕組みにしました

---
## 難しかった点・次回トライしたいこと

- Firebase Authentication と Realtime Database を組み合わせた実装が初めてで、ログイン処理の流れを理解するのに時間がかかりました
- 現在は名前でユーザーを識別しているため、次回はメールアドレス認証に切り替えてより確実に本人を特定できる仕様にしたいです
- クライアント・カウンセラーそれぞれのキャラクターを用意するなど、遊び心のあるデザインにしていきたいです
